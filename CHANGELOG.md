# Changelog

## [Unreleased] — 0.1.0

### Added
- Initial release
- Qwen 3.7-plus (DashScope) multimodal screenshot analysis
- Tiered warning system: toast (1–3) / overlay dialog (4–6) / full-screen blocking overlay (7–10)
- Browser URL capture via Windows UI Automation (Chrome, Edge, Firefox, Brave)
- Clipboard URL monitoring
- System tray integration with pause/resume and scan-now
- First-run setup wizard with privacy disclosure and API key setup
- Windows Credential Manager for secure API key storage
- Autostart via HKCU registry key
- PyInstaller + Inno Setup for no-UAC Windows installer
- GitHub Actions CI (Windows, Python 3.11) and release build workflow
